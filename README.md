# Multithreading in Java

This repository contains laboratory works completed as part of the Multithreading in Java course.

## Laboratory Work 1

Implementation of basic multithreaded task execution using Java concurrency utilities.

## Laboratory Work 2

Simulation of a ferry transportation system with concurrent vehicle processing and state-based ferry operation.

## Laboratory Work 3

Implementation of a currency rate processing system using asynchronous data publishing and subscriber-based event handling.
